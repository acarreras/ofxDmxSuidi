ofxDmxSuidi
===========

Introduction
------------
A Openframeworks wrapper to use the Nicolaudie's library to control their usb-dmx interfaces.

Licence
-------
The code in this repository is available under the [MIT License](https://secure.wikimedia.org/wikipedia/en/wiki/Mit_license).  

Installation
------------
Install the Nicolaudie Siudi 6C driver
Copy the ofxDmxSuidi to your openFrameworks/addons folder.
Create a new project adding the files and paths as indicated in "install.xml"
Copy the "lib/DasHard2006.dll" to your "bin" folder.

Compatibility
------------
Tested with OF v0073
Windows
